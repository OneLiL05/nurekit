JavaScript library for Nure API. Based on [nure-cist-api](https://github.com/nure-dev/nure-cist-api).

## Installation

```shell
npm i nurekit
# yarn add nurekit
# pnpm add nurekit
```

## Usage

```typescript
import { Nurekit } from "nurekit";

const nurekit = new Nurekit();
```
